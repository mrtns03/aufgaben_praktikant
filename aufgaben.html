Herzlich willkommen zu deiner ersten Aufgabe im Bereich HTML, CSS und JavaScript!

Das Ziel dieser Übung ist es, einen Lebenslauf zu erstellen. Es ist wichtig zu betonen, dass es nicht nur einen richtigen Weg gibt, um dies zu tun. Vielmehr gibt es unzählige Möglichkeiten, wie dein Lebenslauf aussehen kann.

Du hast die Freiheit, zu experimentieren und deine eigenen Ideen umzusetzen. Die hier bereitgestellten Beispiele und Tipps sollen dir als Ausgangspunkt dienen, aber du bist ermutigt, kreativ zu sein und deine eigene Lösung zu finden. Nutze verschiedene HTML-Tags und CSS-Eigenschaften, probiere unterschiedliche Layouts aus und finde deinen eigenen Stil.

Hier sind einige Challenges, die dir Schritt für Schritt dabei helfen, die notwendigen Fähigkeiten zu erlernen. Denk daran: Es ist völlig in Ordnung, Fehler zu machen und aus ihnen zu lernen. Der beste Weg, um zu lernen, ist durch Ausprobieren und Experimentieren. Viel Spaß dabei!

Challenge 1: HTML-Grundlagen

Aufgabe: Erstelle eine einfache HTML-Seite mit einer Überschrift, einem Absatz und einer Liste.

Tipp: Nutze die Tags <h1> für die Überschrift, <p> für den Absatz und <ul> und <li> für die Liste.
Mögliche Quellen:
  • W3Schools HTML Tutorial

Challenge 2: CSS-Grundlagen

Aufgabe: Füge einer HTML-Seite eine CSS-Datei hinzu. Ändere die Hintergrundfarbe, die Schriftart und den Textstil der Seite.

Tipp: Verlinke die CSS-Datei im <head> deiner HTML-Datei mit <link rel="stylesheet" href="styles.css">. Verwende CSS-Eigenschaften wie background-color, font-family und color.
Mögliche Quellen:
  • W3Schools CSS Tutorial

Challenge 3: Struktur eines Lebenslaufs

Aufgabe: Erstelle eine HTML-Seite, die die Grundstruktur eines Lebenslaufs enthält (Abschnitte für persönliche Informationen, Berufserfahrung, Ausbildung, Fähigkeiten und Hobbys).

Tipp: Nutze die Tags <header>, <section>, <h2>, <ul> und <li>. Halte die Struktur übersichtlich und konsistent.

Challenge 4: Styling eines Lebenslaufs

Aufgabe: Style die verschiedenen Abschnitte des Lebenslaufs, sodass sie klar und ansprechend aussehen.

Tipp: Verwende Klassen (class) und IDs (id), um spezifische Elemente zu stylen. Experimentiere mit verschiedenen Layout-Techniken wie Flexbox oder Grid.
Mögliche Quellen:
  • CSS Klassen und IDs
  • Grid und Flexbox erklärt

Challenge 5: Hinzufügen eines Bildes

Aufgabe: Füge dem Lebenslauf ein Profilbild hinzu. Verwende das <img>-Tag, um das Bild einzufügen, und style es mit CSS, um es ansprechend zu präsentieren.

Tipp: Nutze das <img />-Tag um ein Profilbild einzubinden. Verwende CSS-Eigenschaften wie width, height und border-radius, um das Bild zu stylen (z.B. als runder Avatar).
Mögliche Quellen:
  • MDN Web Docs - <img> Element
  • CSS-Tricks - The border-radius Property

Challenge 6 (optional): Responsive Design

Aufgabe: Mache den Lebenslauf responsiv, sodass er auf verschiedenen Bildschirmgrößen gut aussieht.

Tipp: Verwende Medienabfragen (@media) in CSS, um das Layout für kleinere Bildschirme anzupassen. Prüfe deine Seite auf verschiedenen Geräten oder mit den Entwicklertools deines Browsers.
Mögliche Quellen:
  • MDN Web Docs - Responsive Design
  • W3Schools - CSS Media Queries

Challenge 7: Dynamische Berufserfahrung

Aufgabe: Füge eine Schaltfläche hinzu, mit der du neue Einträge zur Berufserfahrung hinzufügen kannst.

Tipp: Erstelle eine JavaScript-Datei und binde sie ein.
Mögliche Quellen:
  • W3Schools JavaScript HTML DOM

Challenge 8: Berufserfahrung mit Namen hinzufügen

Aufgabe: Erstelle ein Eingabefeld und eine Schaltfläche, mit denen du neue Einträge zur Berufserfahrung mit einem Namen hinzufügen kannst.

Tipp: Verwende JavaScript, um den Namen aus dem Eingabefeld zu lesen und einen neuen Listeneintrag zu erstellen.
Mögliche Quellen:
  • W3Schools JavaScript Input Examples

Challenge 9: Design und Funktionalitäten verbessern

Aufgabe: Verbessere das Design und füge weitere Funktionalitäten hinzu. Es gibt keine konkreten Vorgaben, probiere am besten alles Mögliche aus, um die Seite benutzerfreundlicher und ansprechender zu gestalten.

Tipp: Du kannst Farben, Layouts, Animationen und weitere interaktive Elemente hinzufügen. Auch das Hinzufügen neuer Funktionen wie das Bearbeiten und Löschen von Berufserfahrungseinträgen kann interessant sein.
Mögliche Quellen:
  • W3Schools CSS
  • W3Schools JavaScript
  • MDN Web Docs
